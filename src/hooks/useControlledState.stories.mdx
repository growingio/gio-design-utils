# useControlledState

当组件既要自己维护 `state`，并且还要根据 `props` 进行更新时，推荐使用 `useControlledState` 来创建 `state`。

## 如何使用

```ts
import React from 'react';
import { Input } from '@gio-design/components';
import { useControlledState } from '@gio-design/utils';

function App({ value }) {
  const [controlledValue, setControlledValue] = useControlledState(value, '');
  return <Input type="text" value={controlledValue} onChange={(e) => setControlledValue(e.target.value)} />;
}
```
